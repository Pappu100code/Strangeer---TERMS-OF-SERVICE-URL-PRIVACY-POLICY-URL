# Privacy Policy for LogCore Discord Bot

**Effective Date:** January 22, 2026  
**Last Updated:** January 22, 2026

## 1. Introduction

LogCore ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our Discord bot service ("LogCore Bot" or "Service").

## 2. Information We Collect

### 2.1 Automatically Collected Information
- **Discord User Data:** User IDs, usernames, discriminators, avatars
- **Server Data:** Guild IDs, server names, channel information, role data
- **Message Data:** Message content, timestamps, attachments (for logging purposes only)
- **Voice Data:** Voice channel join/leave events, voice state changes
- **Moderation Data:** Bans, kicks, timeouts, warnings, and other moderation actions
- **Interaction Data:** Command usage, button clicks, slash command interactions

### 2.2 Server Configuration Data
- **Log Channel Settings:** Configured logging channels and preferences
- **Webhook Information:** Webhook URLs and configurations for log delivery
- **Permission Settings:** Role-based access controls and bot permissions
- **Maintenance Settings:** Maintenance mode configurations and notifications

### 2.3 Technical Data
- **Server Performance:** Bot latency, response times, error logs
- **Usage Analytics:** Command frequency, feature utilization statistics
- **Error Reports:** Bug reports submitted through the bot interface

## 3. How We Use Your Information

### 3.1 Primary Purposes
- **Event Logging:** Monitor and log server activities as configured by administrators
- **Moderation Support:** Assist server moderators with tracking user behavior
- **Bot Functionality:** Provide core logging and server management features
- **Performance Monitoring:** Ensure optimal bot performance and reliability

### 3.2 Secondary Purposes
- **Service Improvement:** Analyze usage patterns to enhance bot features
- **Technical Support:** Troubleshoot issues and provide user assistance
- **Security:** Detect and prevent abuse or malicious activities
- **Compliance:** Meet legal obligations and enforce our Terms of Service

## 4. Information Sharing and Disclosure

### 4.1 We Do Not Sell Your Data
We do not sell, trade, or rent your personal information to third parties.

### 4.2 Limited Sharing Scenarios
- **Server Administrators:** Log data is shared with designated server administrators
- **Legal Requirements:** When required by law, court order, or government request
- **Service Providers:** Trusted third-party services that help us operate the bot
- **Safety Concerns:** To protect users' safety or investigate violations

### 4.3 Data Processing Partners
- **MongoDB Atlas:** Database hosting and management
- **Discord Inc.:** Platform integration and API services
- **Hosting Providers:** Server infrastructure and maintenance

## 5. Data Storage and Security

### 5.1 Storage Location
- Data is stored on secure servers with industry-standard encryption
- Database backups are maintained for disaster recovery purposes
- All data transmission uses encrypted connections (HTTPS/WSS)

### 5.2 Security Measures
- **Encryption:** Data encrypted both in transit and at rest
- **Access Controls:** Strict authentication and authorization protocols
- **Monitoring:** Continuous security monitoring and threat detection
- **Regular Updates:** Security patches and system updates applied promptly

### 5.3 Data Retention
- **Active Servers:** Data retained while bot is active in server
- **Inactive Servers:** Data purged 30 days after bot removal
- **Log Data:** Server logs retained for maximum 90 days unless configured otherwise
- **Error Reports:** Technical logs retained for 1 year for debugging purposes

## 6. Your Rights and Choices

### 6.1 Access Rights
- Request access to your personal data we have collected
- Receive a copy of your data in a portable format
- Inquire about how your data is being processed

### 6.2 Control Rights
- **Data Deletion:** Request deletion of your personal data
- **Data Correction:** Request correction of inaccurate information
- **Processing Restriction:** Limit how we process your data
- **Opt-Out:** Withdraw consent for data processing where applicable

### 6.3 Server Administrator Rights
- Configure logging preferences and data retention periods
- Export server log data for backup purposes
- Delete all server-related data by removing the bot
- Control which events are logged and stored

## 7. Children's Privacy

LogCore is not intended for users under 13 years of age. We do not knowingly collect personal information from children under 13. If we become aware that we have collected personal information from a child under 13, we will take steps to delete such information promptly.

## 8. International Data Transfers
